About Me

As a DevOps Engineer at Thrivent, I am passionate about delivering scalable, reliable, and secure cloud solutions for various business needs. I have over five years of experience in designing, implementing, and automating cloud infrastructure using AWS, Azure, and VMware platforms. I have successfully converted existing AWS infrastructure to serverless architecture using AWS Lambda and Kinesis, and deployed them using Terraform and AWS CloudFormation templates. I have also leveraged Docker, Chef, and Ansible tools to create and manage containerized microservices using Kubernetes. Additionally, I have configured and maintained CI/CD tools such as Git, Jenkins, Nexus, and Jira for multi-environment deployment and orchestration. I am always eager to learn new technologies and best practices to optimize performance, efficiency, and quality of cloud solutions. Areas of Expertise: Specialized knowledge in Serverless and Microservices Architecture. Extensive experience with AWS Services, including IAM, EC2, VPC, S3, CloudWatch, Lambda, API Gateway, SNS, SQS, RDS, EKS, and more. In-depth understanding and utilization of Azure Services, such as Virtual Machines, Virtual Networks, Blob Storage, Data Factory, Data Lake, Databricks, among others. Proficient in implementing DevOps practices with CI/CD pipelines, utilizing tools like Jenkins, Git, Docker, and Kubernetes. Skilled in Infrastructure as Code (IAC) using Terraform and AWS CloudFormation. Accomplished in Project Management, adept at creating compelling Presentations, and proficient in crafting Proposals. Expertise in Cost Estimation and Optimization strategies for cloud resources. Proficient in Capacity Planning and skilled in implementing Data Governance best practices.
